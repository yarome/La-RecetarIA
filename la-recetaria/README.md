# La RecetarIA

La documentación principal del repositorio (la que GitHub muestra en la portada) está en la raíz: **[README.md](../README.md)**.

## Desarrollo local

Desde esta carpeta (`la-recetaria/`):

```bash
npm install
npm run dev
```

Scripts adicionales: `npm run build`, `npm run typecheck`, `npm run seed:users`, `npm run seed:sample-recipes`.
